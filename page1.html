Useful SLURM Cluster Codes

  • Submit a Job, the job_script.sh contains commands and instructions to run a code in cluster
    sbatch job_script.sh

    inside the job_script.sh

    #!/bin/sh
    #SBATCH --job-name=training5
    #SBATCH --N 1
    #SBATCH --n 14 #number of cores
    #SBATCH --gres=gpu:1 ## Run on 1 GPU
    #SBATCH --output new.out
    #SBATCH --error report_error.err
    #SBATCH --p v100-16gb #partition name

    module load python3/anaconda/2021.11
    module load cuda/11.1
    conda activate newALIGNN #load local environment
    python3 train_folder.py

  • Check Job Status

    squeue -u your_username

  • Cancel a Job

    scancel job_id

  • View Job Output

    vi job_id.out

  • Request Interactive Session

    srun -i --pty bash

  • Check Node Information, this shows available nodes in a cluster

    sinfo

  • Check Partition Information

    sinfo -p partition_name

  • Check Queue Information

    squeue -p partition_name

10 Useful SLURM Cluster Codes

  • Submit a Job

    sbatch job_script.sh

  • Check Job Status

    squeue -u your_username

  • Cancel a Job

    scancel job_id

  • View Job Output

    cat job_id.out

  • Request Interactive Session

    srun -i --pty bash

  • Check Node Information

    sinfo

  • Check Partition Information

    sinfo -p partition_name

  • Check Queue Information

    squeue -p partition_name

  • Create a New Partition

    scontrol create partition name=new_partition nodes=1:1,2:1 state=enabled

  • Modify a Partition

    scontrol modify partition name=new_partition state=disabled
